# Megan Cloud Landing Page

صفحه فرود پلتفرم «مگان» با استفاده از React، TypeScript و Vite پیاده‌سازی شده است. طراحی مطابق ماکت ارائه‌شده توسعه داده شده و برای نمایش در نمایشگرهای دسکتاپ و موبایل بهینه‌سازی شده است.

## اجرای پروژه

```bash
npm install
npm run dev
```

## ساخت نهایی

```bash
npm run build
```

خروجی نهایی در پوشه `dist` تولید می‌شود و می‌توان آن را روی هر وب‌سرور ایستا قرار داد.
